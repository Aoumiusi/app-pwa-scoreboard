# スポーツ得点板 PWA

バレーボールや卓球などのスポーツに対応したシンプルな得点板アプリです。Progressive Web App (PWA) として実装されており、モバイルデバイスへのインストールやオフライン利用が可能です。

## 機能

- **セット制スコア管理** — 1〜5セットの試合形式に対応
- **勝利条件** — 25点先取（2点差必要、最大30点）
- **タイマー** — 試合時間の計測（開始・一時停止・リセット）
- **チーム名編集** — タップ/クリックで直接編集可能
- **PWA対応** — ホーム画面への追加・オフライン動作

## 操作方法

### モバイル（タッチ）
| 操作 | 動作 |
|------|------|
| スコア表示を上スワイプ | +1点 |
| スコア表示を下スワイプ | −1点 |
| スコア表示をダブルタップ | セット獲得 |

### PC（マウス・キーボード）
| 操作 | 動作 |
|------|------|
| スコア表示でホイール上 | +1点 |
| スコア表示でホイール下 | −1点 |
| スコア表示をダブルクリック | セット獲得 |
| `Q` / `A` | チームA +1 / −1 |
| `P` / `L` | チームB +1 / −1 |
| `W` / `O` | チームA / B セット獲得 |
| `Space` | タイマー開始・一時停止 |
| `R` | タイマーリセット |

## 開発・動作確認

ビルドツール不要。以下のいずれかでブラウザから開きます。

```bash
# ファイルを直接開く
open index.html

# ローカルサーバー（PWA機能のテスト推奨）
python -m http.server 8000
# または
npx serve .
```

## 技術スタック
